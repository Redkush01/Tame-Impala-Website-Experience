# Tame Impala Website Experience (UI/UX project)

#link: https://redkush01.github.io/Tame-Impala-Website-Experience/

Interactive web-based gestural interface for real-time audio-visual control. Developed using MediaPipe for computer vision and hand tracking, integrated with custom JS logic for low-latency multimedia interaction.
An experience that explores the intersection of gesture control and psychedelic audio-visual feedback.

This project was developed to create a hands-free, immersive interface for manipulating soundscapes in real-time, bridging human motion with digital art.

# The Concept
Using computer vision, the browser tracks your hand movements to act as a controller for sound and visuals. No physical controllers—just your hands and the music of Tame Impala.

# Tech Stack
Computer Vision: MediaPipe Hand Landmarker

Frontend: Vanilla JavaScript, CSS3, HTML5

Audio Processing: Web Audio API

Deployment: GitHub Pages

# Key Features
Real-time Hand Tracking: Precise spatial interaction using MediaPipe.

Low-Latency Interaction: Optimized for seamless audio-visual sync.

Immersive Design: Psychedelic aesthetics inspired by Tame Impala's visual identity.

📂 Project Structure
/js: Core logic for computer vision, audio processing, and UI interactions.

/css: Styles defining the visual experience.

/assets: Audio samples and visual textures.
